# Gym App - Product Requirements Document

## Goal

A simple workout tracking web app for personal use.

## Core Features (V1)

-   User registration and login (JWT)
-   Predefined exercises list (seeded)
-   Create workout
-   Add exercises to workout
-   Add sets (reps + weight)
-   Edit saved workout
-   Workout history
-   Simple rest timer (frontend only)
-   Dark mode toggle (frontend only)

## Non Goals

-   AI suggestions
-   Social features
-   Advanced analytics
-   Programs or templates

## Target Device

Mobile-first (Galaxy S21)

## UX Priorities

-   Large touch buttons
-   Fast input
-   Minimal distraction
